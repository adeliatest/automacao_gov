# Título:
Breadcrumbs - Card de material de apoio
# Resumo:
Caso de teste para verificar se a url do card de material de apoio irá para a página correta pelo histórico de navegação (breadcrumbs)
# Navegador:
Chrome
# Url:
https://www.gov.br/governodigital/pt-br/acessibilidade-digital
# Pré-requisito:
- Entrar na página  de acessibilidade do gov.br
# Passos:
1) Clicar no card com título "Material de apoio"
2) Verificar se o histórico de navegação (breadcrumbs) termina com "Material de apoio"
# Resultado:
O histórico de navegação é igual a "Acessibilidade e Usuário > Acessibilidade Digital > Material de apoio"

# LOGIN 

# CENÁRIO:
Login
# Título:
Login - digitar letras e símbolos no campo de cpf
# Resumo:
Verifica se o campo de cpf aceita que o usuário insira algo além de números
# Navegador:
Chrome
# Url:
https://sso.acesso.gov.br
# Pré-requisito:
Estar na página de login https://sso.acesso.gov.br
# Passos:
1) Clicar no campo de cpf
2) Tentar digitar alguma letra ou símbolo
# Resultado:
O campo deverá continuar vazio, pois não permite inserção de letras
